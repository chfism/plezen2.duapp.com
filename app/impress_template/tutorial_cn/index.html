你现在看到的是本展报(presentation)第一屏。

在做这个展报之前，需要明白以下几点：

  1. 如作者所言，you need to be a designer，否则做出来的展报就会如我这份一样
  2. you have something to show，假如没有，做出来的展报就会如我这份一样
  3. 做这个展报最重要的是，设计与内容，代码的实现只是很小的一步，每屏都只是平常的 HTML + CSS，所以假如你有非常棒的内容，接下来该考验你的设计能力，或者，脑补能力。代码？交给我这样的码人去写就好。

本 div 块的 ID 值为 "welcome"，这样在浏览器中查看展报时，url 地址会显示为 xxx.html#/welcome，假如 div 不带 ID，则 impress.js 会自动赋给 url 一个 "step-N" 的值。当然，ID 名要比 "step-N" 有意义多，而且可以针对其做更多样式。另外，本屏所处的位置由 data-x data-y 值决定，参照点为 canvas 的中心原点。
